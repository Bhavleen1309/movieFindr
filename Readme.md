# MovieFindr

MovieFindr is a small Flask web app that recommends movies. It provides a simple web UI and uses functions in `recommendationFunctions.py` to generate recommendations.

Prerequisites
- Python 3.10+ (or 3.8+)
- Install dependencies from `requirements.txt`

Project layout

- `app.py` — Flask application entry point
- `recommendationFunctions.py` — recommendation logic and helpers
- `templates/index.html` — main HTML template
- `requirements.txt` — Python dependencies
